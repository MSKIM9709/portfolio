# 🎨 김민수 포트폴리오 디자인 & 콘텐츠 가이드라인 (AI/Developer Rules)

이 문서는 김민수 님의 포트폴리오를 업데이트하거나 새로운 페이지/컴포넌트를 추가할 때, 불필요한 수정 반복 과정 없이 **처음부터 김민수 님의 취향과 기준에 최적화된 프리미엄 결과물**이 도출되도록 정의한 규칙 파일입니다.

이 디렉토리에서 작업을 수행하는 모든 AI 어시스턴트와 개발자는 **반드시 아래 지침을 준수하여 결과물을 즉각 도출**해야 합니다.

---

## 1. 🎨 시각적 디자인 가이드 (Visual Design Principles)

### 🅰️ 폰트 및 안티앨리어싱 규칙 (Font & Typography)
* **기본 폰트**: 한글 렌더링이 가장 매끄럽고 트렌디한 **Pretendard(프리텐다드)**를 최우선 폰트로 사용합니다.
* **테크 모노스페이스 폴백 설정**:
  * 영어와 숫자를 모노스페이스 서체(`JetBrains Mono`)로 표현할 때, 한글이 섞여 있을 경우 시스템 기본 모노스페이스(굴림체 등)로 깨져 렌더링되지 않도록 반드시 아래 폴백 체계를 유지합니다.
  * `font-family: 'JetBrains Mono', 'Pretendard', monospace;`
* **텍스트 렌더링 최적화**: 윈도우 환경에서도 글꼴 경계선이 자글자글하지 않도록 `body`에 다음 안티앨리어싱 속성을 필수로 적용합니다.
  ```css
  body {
      -webkit-font-smoothing: antialiased;
      -moz-osx-font-smoothing: grayscale;
      text-rendering: optimizeLegibility;
  }
  ```

### 📐 긴밀하고 정교한 간격 설계 (Compact Layout)
* **적절한 컴포넌트 간격**: 스크롤 시 화면이 지나치게 비어 보이거나 뚝뚝 끊겨 보이는 것을 방지하기 위해 섹션 간 상하 여백(Padding)은 **`60px 0`** 수준으로 긴밀하게 설계합니다.
* **구분선 활용**: 무작정 공간을 늘리는 것보다, 섹션 경계부에 은은한 그레이 톤의 얇은 점선(`border-top: 1px dashed rgba(255, 255, 255, 0.06);`)이나 미니멀한 시각 지표를 삽입하여 구분을 명확히 합니다.

### 💼 비즈니스 미니멀리즘 및 이모지 배제 (No Emojis)
* **이모지 최소화**: 포트폴리오의 전문성과 차분함을 훼손하는 운영체제 기본의 알록달록한 만화풍 이모지(💼, 📊, 📈, ⚙️ 등)를 카드 내부의 대형 아이콘이나 배지로 활용하는 것을 전면 배제합니다.
* **대안**: 기하학적인 CSS 라이트 효과, 얇은 선으로 구성된 테크니컬 디자인 칩, 미니멀한 구분 기호나 특수 기호(✓, → 등)로 단정하게 연출합니다.

### 📐 가로 와이드 카드의 황금 비례 (Horizontal Grid Split)
* **공백 최소화**: 카드가 가로로 넓어질 경우, 한쪽에만 텍스트가 쏠려 반대편에 불필요한 거대 공백이 생기지 않도록 차단합니다.
* **2분할 그리드**: 카드 내부에 좌측(핵심 인덱스/대시코드/타이틀)과 우측(세부 설명 목록)을 나누는 **수평 분할 그리드(`grid-template-columns: 0.95fr 1.05fr;`)**를 적극적으로 사용하여 균형을 맞춥니다.

## 2. 📝 콘텐츠 및 톤앤매너 가이드 (Content & Tone Guide)

### 🇰🇷 철저한 국문(한글) 중심 표기
* **직관적인 한글화**: 본문의 라벨, 버튼, 설명, 내비게이션 등 주요 메뉴에는 불필요한 영문 테크 단어나 생소한 외래어 배지(예: `[SYS.OPS]`, `[DAT.ANL]`)를 남발하지 않고, 신뢰감을 주는 **바른 한글 어휘**를 최우선으로 사용합니다.

### 📊 객관적인 사실 기반 서술 (Factual & Objective Tone)
* **주관적 미사여구 배제**: 스스로의 역량을 직접 수식하는 표현인 **'능숙하게'**, **'완벽하게'**, **'성공적으로'** 등 자화자찬식 부사나 형용사를 완전히 삭제합니다.
* **수치 기반 정량화**: "데이터를 능숙하게 정제하여 인건비를 시뮬레이션했다"가 아니라, "**기술직 470명의 5개월 대용량 실데이터를 정제하여 통상임금 시뮬레이션을 설계했다**"와 같이 철저히 사실(Fact)과 정량 수치(Metric) 중심의 문장으로 구성합니다.

### 🌟 핵심 업무 철학의 공간적 쇼케이스화
* **인사 철학 강조**: 김민수 님의 핵심 업무 가치관인 **"인사의 감과 경험에 데이터 분석의 타당성을 입혀 최적의 답을 도출합니다."** 격언 카드는 포트폴리오의 얼굴과 같으므로 아래 디자인 가이드를 필히 준수합니다.
  * 큼직한 반투명 큰따옴표(`“`) 배경 이펙트 배치.
  * 텍스트 폰트 사이즈는 대형(`1.55rem`), 두꺼운 볼드(`80px`), 빛나는 그라데이션 필터 적용.
  * 아래쪽에 이를 뒷받침하는 깔끔한 한글 캡션 매칭.
